# State deep-sweep tracker

Goal: bring every state to florida-cyber-map depth, one state at a time.
Statuses: **anchors** (majors only) → **pass 1** (web-verified sweep of
cons/meetups/CTFs/makers/schools/libraries/youth) → **deep** (Florida-level:
addresses, phones, schedules, county library systems).

## Per-state sweep checklist (repeat for each state)

1. Cons: every BSides + regional cons — verify site + next dates
2. Meetups: DEF CON groups, 2600, OWASP/ISSA/ISACA/ISC2 chapters, local
   hacker associations — meeting day/time/venue
3. CTFs: student-run CTFs, CCDC/CPTC regional hosts, state cyber games
4. Makerspaces/hackerspaces: status, address, open-house nights
5. Schools: CAE list for the state + active student clubs/CTF teams
6. **Library systems: makerspace/STEM programs (check the big city +
   county systems, not just one branch)**
7. Youth STEM: GenCyber host sites, CyberPatriot hotbeds, FIRST hubs,
   state programs
8. Gov/research: bases, labs, fusion centers, state cyber offices
9. Validate (`st` codes, point-in-state), commit
